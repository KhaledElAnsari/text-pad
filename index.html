Text-Pad

A tiny solution to add some white spaces to your text

Ever wanted to add some white space to your string with Javascript? if yes you're at the right place (I guess).

This project is so tiny (and useless for some people including me) but after reading about left-pad problem with npm I was like "WOW! even useless stuff can get some attention, let me try something like it but a lil bit better" and here I am releasing this project.

Text-Pad is enable to do 4 types of string padding. to see how to use Text-Pad go to the project github page, to see it in action scroll down to see the examples
